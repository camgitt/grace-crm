# Header Background Images

Place stock images for page headers in this folder.

## Expected Files
- `dashboard.jpg` - Dashboard page header
- `people.jpg` - People directory header
- `giving.jpg` - Giving/Donations header
- `tasks.jpg` - Tasks page header
- `calendar.jpg` - Calendar/Events header
- `groups.jpg` - Groups page header
- `reports.jpg` - Reports page header
- `settings.jpg` - Settings page header
- `integrations.jpg` - Integrations page header
- `life-apps.jpg` - LifeApps/Devotionals header
- `accounting.jpg` - Accounting page header

## Recommended Specifications
- **Format**: JPG or WebP
- **Size**: 1920x400 pixels minimum
- **Style**: Church/community themed, soft focus works well
- **Note**: Images will have a gradient overlay, so lighter images work best

## Sources for Free Stock Images
- [Unsplash](https://unsplash.com) - Search "church", "community", "worship"
- [Pexels](https://pexels.com) - Search "congregation", "prayer", "fellowship"
- [Pixabay](https://pixabay.com) - Search "church interior", "community gathering"
